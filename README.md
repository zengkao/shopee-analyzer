# 蝦皮訂單 & 廣告費分析工具

讓蝦皮賣家可以用自己的數據，分析營業額、廣告費、優惠券、平台抽成的趨勢。

**讓數據說話，拒絕黑箱。**

---

## 這個工具能做什麼？

從蝦皮後台匯出的原始報表，自動計算：

- 各月營業額、訂單數、買家實付金額
- 優惠券使用率、蝦皮負擔優惠券金額、蝦幣折抵
- 成交手續費、其他服務費、金流處理費（平台費率）
- 廣告花費、ROAS、CPC、轉換率
- 總成本（平台費 + 廣告費）與實際到手估算
- 月度變化趨勢比較

## 使用方式

### 方法一：直接下載 exe（不需安裝 Python）

1. 到 [Releases](../../releases) 頁面下載 `蝦皮分析工具.exe`
2. 準備資料：
   - 建立一個資料夾，裡面放兩個子資料夾：
     - `訂單/` — 放蝦皮後台匯出的訂單 Excel 檔（.xlsx）
     - `廣告費/` — 放蝦皮後台匯出的廣告數據 CSV 檔（.csv）
3. 執行 `蝦皮分析工具.exe`
4. 輸入資料夾路徑（或直接按 Enter 使用目前目錄）
5. 如果訂單檔有密碼，輸入密碼
6. 等待分析完成，報告會顯示在畫面上並儲存為 txt 檔

### 方法二：使用 Python 原始碼

```bash
# 安裝依賴
pip install pandas openpyxl msoffcrypto-tool

# 執行
python shopee_analyzer.py
```

## 如何從蝦皮後台匯出資料

### 匯出訂單
1. 登入蝦皮賣家中心
2. 進入「訂單管理」→「我的訂單」
3. 選擇日期區間
4. 點擊「匯出」，選擇 Excel 格式

### 匯出廣告數據
1. 登入蝦皮賣家中心
2. 進入「行銷中心」→「我的廣告」
3. 選擇日期區間
4. 點擊「匯出報表」

## 為什麼做這個工具？

2025年10月蝦皮推出「智慧優惠券」功能後，許多賣家發現廣告費暴漲、ROAS 下降、營業額反而下滑。

蝦皮官方聲稱「賣家僅支付廣告點擊費用，不額外支付優惠券成本」，但同時也說「買家使用優惠券下單時，賣家收到的訂單金額為折扣後金額」。

**一個人的數據是個案，一百個人的數據就是證據。**

歡迎所有蝦皮賣家下載使用，分析自己的數據，一起讓真相浮出水面。
